# 📊 Excel Data Analysis – IBM Professional Certificate

This repository contains my Excel-based data analysis project completed as part of the **IBM Data Analyst Professional Certificate**. The project demonstrates practical data cleaning, transformation, analysis, and visualization using raw CSV and Excel datasets.

---

## 📌 Project Overview

In this project, I worked with raw datasets and performed structured data analysis using Microsoft Excel. The objective was to clean the data, explore trends, and extract meaningful business insights using formulas, pivot tables, and visualizations.

This project reflects hands-on experience with real-world data preparation and analysis workflows.

---

## 🛠 Skills Demonstrated

* Data cleaning and preparation
* Handling raw CSV and Excel files
* Sorting and filtering large datasets
* Removing duplicates and handling missing values
* Using Excel formulas (e.g., IF, SUMIFS, COUNTIFS, VLOOKUP/XLOOKUP)
* Creating pivot tables for summary analysis
* Building charts and visualizations
* Extracting business insights from structured data
